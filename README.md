# Community Board

A full-stack Community Board application built with React (Vite), Node.js (Express), and MongoDB. Users can register, login, create posts, view posts, update their posts, and delete posts.

Features

User Authentication

Register new users

Login and logout

JWT-based protected routes

Post Management

Create, Read, Update, Delete posts

Only post owners can update or delete their posts

Posts display author and creation date

Frontend

Built with React and Vite

Responsive and clean UI

Dark/light mode support

Backend

Node.js with Express

MongoDB database with Mongoose

RESTful API routes for auth and posts

Tech Stack

Frontend: React, Vite, React Router, Axios

Backend: Node.js, Express, JWT, bcryptjs

Database: MongoDB (Atlas)

Styling: CSS, Responsive Design

Installation
Prerequisites

Node.js >= 20.19

npm

MongoDB Atlas account or local MongoDB instance
